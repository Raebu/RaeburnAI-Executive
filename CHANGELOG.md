# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-02

### Added

- Production-oriented FastAPI briefing API.
- Next.js executive dashboard.
- Typed briefing, KPI, signal and action models.
- Mock connectors for email, calendar, KPI, sales, risk, competitor and news signals.
- Health and readiness endpoints.
- API key guard, input validation, rate limiting and human approval gate for risky actions.
- Unit, integration and E2E smoke tests.
- Dockerfiles and docker-compose configuration.
- CI, CodeQL, dependency review and Dependabot configuration.
- Standard RaeburnAI README, deployment, security, contribution and screenshot docs.
